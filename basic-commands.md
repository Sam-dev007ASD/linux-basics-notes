# Basic Linux Commands

## Navigation
- pwd  → show current directory
- ls   → list files
- cd   → change directory

## File operations
- touch file.txt   → create file
- mkdir test       → create folder
- cp a b           → copy
- mv a b           → move / rename
- rm file.txt      → delete file

## Viewing files
- cat file.txt
- less file.txt
- head file.txt
- tail file.txt
